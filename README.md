# AI-Powered Energy Consumption Forecasting System ⚡

This project is a comprehensive, industry-aligned machine learning application designed to forecast electricity usage in smart cities and industrial environments.

## 🌟 Live Preview
The application features a real-time React dashboard that simulates the energy forecasting workflow, including:
- **Synthetic Data Generation**: Simulating smart grid logs with human behavioral patterns.
- **Neural Network Simulation**: Visualizing the training and inference of an MLP Regressor.
- **Interactive Visualizations**: Real-time Actual vs. Forecasted energy trends using Recharts.

## 📁 Project Structure
- `/src`: React frontend dashboard and simulation engine.
- `/python_project`: Complete, GitHub-ready Python implementation.
  - `main.py`: Data generation, preprocessing, and model training.
  - `app.py`: Flask API for real-time predictions.
  - `requirements.txt`: Python dependencies.
  - `README.md`: Professional documentation for your GitHub repository.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Recharts, Motion.
- **Backend (Simulation)**: TypeScript-based behavioral modeling.
- **AI (Python)**: Scikit-Learn (MLPRegressor), Pandas, NumPy.
